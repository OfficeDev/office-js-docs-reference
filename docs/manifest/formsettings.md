# FormSettings element

Specifies source location and control settings for your mail add-in in older Outlook clients.

**Add-in type:** Mail

## Syntax

```XML
<FormSettings>
    <Form xsi:type="ItemRead">
        ...
    </Form>
</FormSettings>
```

## Contained in

- [OfficeApp](officeapp.md)

## Can contain

- [Form](form.md)
