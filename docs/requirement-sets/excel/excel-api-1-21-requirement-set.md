# What's new in Excel JavaScript API 1.21

The ExcelApi 1.21 supports programmatically loading a linked entity cell value and automatically running an event handler when loading completes. To learn more about loading linked entity cell values, see [Query linked entity cell values](/office/dev/add-ins/excel/excel-data-types-linked-entity-cell-values#query-linked-entity-cell-values).
